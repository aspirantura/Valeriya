БЛОК-СХЕМА ДИЗАЙНА ИССЛЕДОВАНИЯ

Проспективное одноцентровое когортное исследование с элементами «случай-контроль»

🎯 Ключевая цель: Разработка прогностической математической модели для расчёта индивидуального риска АГЭ на основе комбинации метаболических (HOMA-IR, лептин) и онкологических (HE4, индекс ROMA) сывороточных маркеров с дальнейшим внедрением дифференцированного клинического алгоритма ведения пациенток.

СКРИНИНГ И ОТБОР

Женщины 35–55 лет, находящиеся на обследовании и лечении в гинекологическом и эндокринологическом отделениях

✅ КРИТЕРИИ ВКЛЮЧЕНИЯ

  • Возраст 35–55 лет
  • Верифицированный МС по IDF (2005)
  • ОТ ≥ 80 см
  • Морфологически подтверждённый диагноз (≤ 3 мес.)
  • Информированное согласие

❌ КРИТЕРИИ НЕВКЛЮЧЕНИЯ

  • Рак эндометрия или др. онкология
  • Гормональная терапия за 6 мес.
  • Беременность, лактация
  • Декомпенсированная патология
  • Острые/обострение хр. заболеваний
  • ВМС < 12 мес., операции на матке

ОБОСНОВАНИЕ РАЗМЕРА ВЫБОРКИ

Мощность (1-β) = 80%  •  α = 0,05 (двусторонний)  •  Ожидаемый эффект OR = 3,0
Формирование групп: сплошная выборка → рандомизированный отбор в контрольную группу
Итого: 45 × 3 = 135 пациенток

ПРОГРАММА ИССЛЕДОВАНИЯ

1Формирование групп и базовое обследование

  • Получение информированного добровольного согласия
  • Стандартизированный сбор анамнеза
  • Антропометрия: рост, масса, ИМТ, ОТ, ОБ
  • Трансвагинальное УЗИ органов малого таза
  • Измерение АД, оценка соматического статуса

2Лабораторное обследование

Венозная кровь натощак, единая лаборатория

  • Метаболический профиль: глюкоза, инсулин (HOMA-IR), липидный спектр
  • Адипокины + воспаление: лептин, адипонектин, hs-CRP
  • Онкомаркеры: HE4, CA-125 (индекс ROMA)
  • Гормональный профиль: эстрадиол, тестостерон, ГСПГ (ИСА), ФСГ, ЛГ

3Статистический анализ и моделирование

  • Проверка данных на нормальность (Шапиро-Уилка)
  • Сравнение групп: ANOVA / критерий Краскела-Уоллиса
  • Оценка корреляций: Спирмена / Пирсона
  • Многомерная логистическая регрессия (пошаговый метод)
  • Внутренняя валидация (bootstrap, 1000 итераций)
  • ROC-анализ: AUC, Se, Sp, PPV, NPV
  • Создание номограммы для расчёта риска АГЭ

4Анализ результатов и внедрение

  • Интерпретация результатов
  • Подготовка статей в журналах ВАК и Scopus
  • Разработка методических рекомендаций
  • Внедрение калькулятора риска АГЭ в клиническую практику
  • Защита диссертации
ПЛАН СТАТИСТИЧЕСКОГО АНАЛИЗА

1Описательная статистика

  • M ± SD (нормальное распределение)
  • Me [Q₁; Q₃] (ненормальное)
  • n, % (категориальные)
  • Шапиро-Уилка

2Сравнительный анализ 3 групп

  • ANOVA (нормальное распределение)
  • Краскела-Уоллиса (ненормальное)
  • Пост-хок: Тьюки / Данна с Бонферрони
  • χ² Пирсона / Фишер

3Корреляционный анализ

  • Коэффициент Пирсона (r)
  • Коэффициент Спирмена (ρ)
  • HOMA-IR ↔ Лептин ↔ hs-CRP
  • Метаболические ↔ HE4, ROMA

4Логистическая регрессия

  • Пошаговый отбор предикторов
  • OR (отношение шансов) + 95% ДИ
  • Коэффициенты модели (β)
  • Тест Хосмера-Лемешова

5ROC-анализ

  • ROC-кривые для каждого маркера
  • AUC + 95% ДИ
  • Cut-off (индекс Юдена)
  • Se, Sp, PPV, NPV, LR+, LR−

6Bootstrap-валидация

  • 1000 итераций
  • Оценка стабильности модели
  • Корректировка optimism
  • Calibration plot

7Структурное моделирование

  • Path analysis (анализ путей)
  • Подтверждение патогенетической цепочки
  • Прямые + опосредованные эффекты
  • Индексы согласия (CFI, RMSEA)

🎯 ПРОГНОСТИЧЕСКАЯ МОДЕЛЬ И НОМОГРАММА

Целевые показатели качества:
AUC > 0,85 (обучающая выборка) • AUC > 0,80 (валидация)

Пример уравнения логистической регрессии:

Логарифм шансов (АГЭ) = 1,2 + 0,8 × (HOMA-IR) + 0,05 × (Лептин)
                      + 0,9 × (ROMA) − 0,03 × (Адипонектин)

💻 ПО

IBM SPSS Statistics 26.0  •  R (pROC, rms)  •  MedCalc
Уровень значимости: p < 0,05

ДИФФЕРЕНЦИРОВАННЫЙ КЛИНИЧЕСКИЙ АЛГОРИТМ
🎯 Принцип персонализации: На основе расчётного индивидуального индекса риска АГЭ пациентка относится к одной из трёх групп с соответствующей тактикой ведения.

🟢 НИЗКИЙ РИСК < 5%

Тактика: Консервативная гормональная терапия под контролем

  • Гормональная терапия (гестагены, ВМС с левоноргестрелом)
  • Динамическое наблюдение каждые 3–6 месяцев
  • Контрольное УЗИ
  • Коррекция метаболических нарушений
  • Сохранение репродуктивного потенциала

🟡 СРЕДНИЙ РИСК 5 – 30%

Тактика: Обязательная гистероскопия с прицельной биопсией

  • Диагностическая гистероскопия с биопсией
  • Морфологическое исследование биоптатов
  • Выбор тактики по результатам
  • Наблюдение каждые 3 месяца

🔴 ВЫСОКИЙ РИСК > 30%

Тактика: Приоритет радикального хирургического лечения

  • Гистерэктомия (тотальная или с придатками)
  • Интраоперационное гистологическое исследование
  • Консультация онколога-гинеколога
  • Онкологическая безопасность приоритетна
